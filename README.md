# Application Project

### Documentation

Consist of **5** modules:

1. msvc-cloud-config-server
2. msvc-naming-service
3. msvc-entity
4. msvc-address
5. msvc-gateway-service

Each module has its own documentation on HELP.md file. For getting the complete application up and running the microservices must be executed in the specified order to avoid startup errors.  

### Requisites

* Java 17 installed
* Maven installed

### REST Service

The entry point of this project is at:
> http://localhost:8080/entities
